# SidePanel UI 重构与视觉升级

## 📋 任务信息
- 类型：🟡 中等任务 (M)
- 预计时间：1 小时
- 目标：提升 SidePanel 的视觉质感，采用现代极简风格，优化阅读体验
- 状态：已完成

## 🎯 方案 (方案 A：极简卡片风)
1.  **色彩升级**：
    - 引入 `Zinc` 色系代替硬编码的黑灰色，提升深色模式质感。
    - 浅色模式优化为 `Zinc-50` 背景 + 白色卡片。
2.  **布局优化**：
    - **移除左侧时间线**：释放水平空间，让内容展示更完整。
    - **卡片重构**：增加卡片间距，使用更柔和的圆角和边框。
    - **头部优化**：添加毛玻璃效果 (Backdrop Blur)，固定顶部。
3.  **细节打磨**：
    - 优化标签 (Tag) 样式。
    - 优化字体行高与层级。
