# 松鼠收藏夹 隐私权政策

**最后更新日期：2024年12月4日**

感谢您使用「松鼠收藏夹」（以下简称"本扩展"）。我们非常重视您的隐私。本隐私权政策旨在向您说明我们如何收集、使用和保护您的数据。

---

## 📋 概述

**松鼠收藏夹** 是一款浏览器扩展，帮助用户收藏 Twitter/X 和小红书上的内容，并提供 AI 辅助摘要和创作功能。

**核心原则**：
- ✅ 所有数据存储在您的浏览器本地
- ✅ 我们不会收集、上传或共享您的个人数据
- ✅ AI 功能由您自行配置的第三方 API 提供

---

## 🔐 数据收集与存储

### 我们收集什么数据？

本扩展**不会主动收集**您的任何个人信息。以下数据仅存储在您的浏览器本地：

| 数据类型 | 存储位置 | 用途 |
|---------|---------|------|
| 收藏的内容 | 浏览器本地存储 | 显示您的收藏列表 |
| AI 摘要结果 | 浏览器本地存储 | 显示内容摘要 |
| API 配置（密钥、URL） | 浏览器本地存储（加密） | 调用您配置的 AI 服务 |
| 用户偏好设置 | 浏览器本地存储 | 保存您的设置选项 |
| 向量索引数据 | 浏览器 IndexedDB | 支持语义搜索功能 |

### 我们不收集什么数据？

- ❌ 个人身份信息（姓名、邮箱、电话等）
- ❌ 浏览历史记录
- ❌ 登录凭据
- ❌ 设备信息
- ❌ 位置数据

---

## 🌐 第三方服务

### AI 服务调用

本扩展支持用户配置第三方 AI 服务（如 OpenAI、阿里云通义千问等）。当您使用 AI 功能时：

1. **您的数据将直接发送到您配置的 AI 服务提供商**
2. 本扩展不会代理或存储这些请求
3. 请参阅您所使用的 AI 服务提供商的隐私政策

**常见 AI 服务提供商的隐私政策：**
- [OpenAI 隐私政策](https://openai.com/privacy)

### 飞书 API（可选）

如果您启用了飞书同步功能，扩展会与飞书 API 通信。请参阅 [飞书隐私政策](https://www.feishu.cn/privacy)。

---

## 🔒 数据安全

我们采取以下措施保护您的数据：

1. **本地存储**：所有数据存储在您的浏览器本地，不上传到任何服务器
2. **加密存储**：API 密钥等敏感信息使用浏览器的加密存储机制
3. **最小权限**：仅申请必要的浏览器权限

### 扩展权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 在浏览器本地保存收藏内容和设置 |
| `sidePanel` | 显示收藏库和创作面板的侧边栏 |
| `tabs` | 获取当前标签页信息以支持内容收集 |

---

## 👤 用户权利

您对自己的数据拥有完全控制权：

- **访问**：您可以随时在扩展侧边栏查看所有收藏内容
- **导出**：支持将数据导出为 JSON 或 Markdown 格式
- **删除**：您可以随时删除单条收藏或清空所有数据
- **卸载**：卸载扩展将自动删除所有本地存储的数据
